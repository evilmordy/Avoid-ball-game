# 躲避小球

双击 index.html 即可在浏览器游玩。

## 操作

- **P1**: 方向键移动
- **P2**: W A S D
- **暂停**: 空格键 / 右上角暂停按钮
- **触屏**: 单指拖拽摇杆

## 球种说明

| 颜色 | 效果 |
|------|------|
| 红色 | 扣 1 滴血（初始 5 HP），HP 归零则游戏结束 |
| 追踪(橙) | 缓慢跟踪玩家，触碰扣血，效果同红球 |
| 分裂(紫) | 数秒后分裂为 3 个小红球，触碰扣血 |
| 金色 | +10 分 |
| 绿色 | +60 分，玩家体型增大 |

## 道具说明

| 图标 | 效果 | 持续时间 |
|------|------|----------|
| S (蓝) | 护盾，免疫红球伤害 | 5 秒 |
| Z (黄) | 加速 1.5 倍 | 4.5 秒 |
| V (绿) | 立即缩小体型 | 瞬间 |

## 模式

| 模式 | 说明 |
|------|------|
| 单人模式 | 经典模式，存活拿分 |
| 双人模式 | P1 + P2 同屏合作 |
| 限时挑战 | 60 秒倒计时，存活即胜利 |

## 难度

| 难度 | 球速 | 生成间隔 | 绿球成长 |
|------|------|----------|----------|
| 简单 | 慢 | 长 | 1.1x |
| 常规 | 中 | 中 | 1.2x |
| 困难 | 快 | 短 | 1.3x |

## 连击系统

连续吃金/绿球累积 combo 数，每 5 连击倍率 +1，最高 x5。中断条件：
- 被红/追踪/分裂球击中
- 超过 2.2 秒未收集

## 技术说明

- 纯 Vanilla JS (ES6)，零依赖
- Canvas 2D 渲染
- 触屏支持：虚拟摇杆 + 暂停按钮
- 窗口自适应缩放

This project was built to test DeepSeek V4 Pro capabilities.
